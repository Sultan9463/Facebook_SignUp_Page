# Facebook SignUp Page

##  Objectifs
Ce projet consiste à réaliser une page d'inscription inspirée de Facebook en utilisant uniquement HTML et CSS. L'objectif est de reproduire l'interface utilisateur de la page d'inscription de Facebook, en mettant l'accent sur la structure sémantique, l'accessibilité et le design responsive.
## Structure du projet  Facebook SignUp Page

## Entete 
- Qui contiendra le titre : 
    -  Creer un compte 
    - rapide et facile
## Body (corps du travail) 
- Il est constitue de :
    -  Nom et prenom
    - adress email et mot de passe
    - date de nassance avec la quelle il ya le choix de :
        - jour
        - moi
        - année
    - le  choix de genre:
        - homme 
        - femme
        - personalise
    - et un texte de condition confidentialité politique
    - en fin un bouton pour s'inscrire
### Technologies utilisées
- HTML5
- CSS3
## Competences aquises: 
Structuration de la page d'inscription <br>
Gérer la responsivité avec les media queries
